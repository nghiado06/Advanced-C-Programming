# TÌM HIỂU VỀ MEMORY LAYOUT
- Tìm hiểu về cách bộ nhớ được tổ chức và sử dụng trong quá trình thực thi chương trình.
- Tìm hiểu về các khái niệm sau:
  - Text (Code) Segment
  - Data Segment (Initialized Data)
  - BSS Segment (Uninitialized Data)
  - Stack

  - Memory Leak
  - Stack Overflow
